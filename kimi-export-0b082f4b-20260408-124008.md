# 🧠 费曼学习法（Feynman Technique）

> **一句话理解**：如果你能用简单的话把一个知识讲给小白听，让他听懂，你才真的懂了。

---

## 📌 核心思想

**"以教促学"** —— 通过教别人来发现自己哪里没懂，然后针对性补强。

> 💡 费曼说："如果你不能向一个6岁小孩解释清楚一件事，说明你还没真正理解它。"

---

## 🎯 四个步骤

| 步骤 | 做什么 | 关键动作 |
|:---:|:---|:---|
| **① 选概念** | 确定要学的内容 | 选一个你想学的知识点（如：牛顿第二定律） |
| **② 教给别人** | 假装在教一个小学生 | 用**最简单的语言**写出来或讲出来，不用术语 |
| **③ 发现盲点** | 卡住的地方就是漏洞 | 哪里讲不通？回去查资料、重新学 |
| **④ 简化类比** | 用比喻让概念更形象 | 比如把"电流"比作"水流" |

---

## 🌰 举个例子

**学"边际效用递减"这个经济学概念：**

- ❌ 术语版：每增加一单位消费，带来的效用增量逐渐减少
- ✅ 费曼版：吃第一个汉堡很香，第二个还行，第三个就腻了，第四个可能吃不下——每多吃一个带来的满足感越来越少

---

## ✅ 什么时候用？

- 学完觉得懂了，但做题不会做
- 准备考试复习
- 给别人讲题时发现自己也卡壳
- 学概念性强的科目（物理、经济、编程等）

# 🍅 番茄工作法（Pomodoro Technique）

> **一句话理解**：设个25分钟闹钟，期间只干一件事，铃响了就休息5分钟，循环往复。

---

## 📌 为什么叫"番茄"？

因为创始人西里洛用的是一个**番茄形状的厨房计时器**（意大利语 Pomodoro = 番茄）。

---

## 🎯 标准操作流程

```
┌─────────────────────────────────────────┐
│  1. 选任务 → 确定今天要做的具体事情        │
│                                         │
│  2. 设闹钟 → 定时 25 分钟                 │
│                                         │
│  3. 专心做 → 直到闹钟响，期间不干别的      │
│                                         │
│  4. 短休息 → 休息 5 分钟（彻底放松）       │
│                                         │
│  5. 循环   → 每完成 4 个番茄钟，长休15-30分 │
└─────────────────────────────────────────┘
```

---

## ⚠️ 三条铁律

| ❌ 不要做 | ✅ 要做 |
|:---|:---|
| 番茄钟内刷手机、回消息 | 25分钟只做一件事 |
| 被打断时暂停继续 | 被打断就作废这个番茄，重新开始 |
| 休息时还想工作 | 休息时彻底放空，喝水、远眺、散步 |

---

## 🌰 实际例子

**写作业场景：**
- 🍅 第1个番茄：写完数学选择题
- ☕ 休息5分钟：喝水、伸懒腰
- 🍅 第2个番茄：写完数学大题
- ☕ 休息5分钟
- 🍅 第3个番茄：背英语单词
- ☕ 休息5分钟
- 🍅 第4个番茄：整理笔记
- 🌴 **长休息20分钟**：玩手机、吃零食

---

## 💡 变体（如果觉得25分钟太长/太短）

| 类型 | 工作+休息 | 适合谁 |
|:---|:---|:---|
| 入门版 | 15分钟 + 5分钟 | 初学者、小学生 |
| 标准版 | 25分钟 + 5分钟 | 大多数人 |
| 深度版 | 45分钟 + 10分钟 | 需要深度思考的任务 |
